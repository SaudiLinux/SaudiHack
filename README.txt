🚨 أداة فحص الثغرات الأمنية للمواقع السعودية
==============================================

📋 وصف المشروع:
أداة متكاملة لفحص وتحليل الثغرات الأمنية في المواقع السعودية باستخدام تقنيات Google Dorking والمسح الأمني.

🔧 متطلبات التشغيل:
- Python 3.8 أو أحدث
- نظام Windows/Linux/Mac
- اتصال إنترنت نشط

📥 خطوات التثبيت:
-----------------

1. **تحميل المشروع:**
   ```
   git clone [رابط المشروع]
   cd SaudiHack
   ```

2. **تثبيت المتطلبات:**
   ```
   pip install -r requirements.txt
   ```

3. **التحقق من التثبيت:**
   ```
   python --version
   pip list
   ```

🚀 طريقة الاستخدام:
------------------

### **الاستخدام السريع:**

1. **عرض قائمة المواقع السعودية:**
   ```
   python saudi_sites_list.py
   ```

2. **فحص موقع محدد:**
   ```
   python google_dork_tool.py --domain example.sa
   ```

3. **عرض المواقع المصابة:**
   ```
   python show_infected_sites.py
   ```

4. **ملخص الأمان السريع:**
   ```
   python security_summary.py
   ```

### **الاستخدام المتقدم:**

1. **فحص فئة معينة من المواقع:**
   ```
   python saudi_scanner.py --category government
   python saudi_scanner.py --category banking
   ```

2. **تشغيل جميع الأدوات:**
   ```
   python run_all.py
   ```

3. **استخدام الواجهة الرسومية:**
   ```
   python main.py
   ```

📊 الأوامر المتوفرة:
-------------------

| الأمر | الوصف |
|-------|---------|
| `python google_dork_tool.py --domain site.sa` | فحص موقع محدد |
| `python saudi_sites_list.py` | عرض قائمة المواقع السعودية |
| `python show_infected_sites.py` | عرض المواقع المصابة |
| `python security_summary.py` | ملخص الأمان |
| `python run_all.py` | تشغيل كل الفحوصات |
| `python run_all_simple.py` | تشغيل مبسط |
| `python main.py` | الواجهة الرسومية |

📁 الملفات المهمة:
-----------------

- **google_dork_report_*.json** - تقارير فحص المواقع
- **saudi_sites.txt** - قائمة المواقع السعودية
- **security_summary_*.txt** - ملخصات الأمان
- **infected_sites_report_*.json** - تقارير المواقع المصابة

⚠️ ملاحظات مهمة:
---------------

1. **الاستخدام القانوني:**
   - استخدم الأداة فقط على المواقع التي تمتلك صلاحية فحصها
   - احترم قوانين البلد وخصوصية المستخدمين

2. **الأمان:**
   - لا تشارك تقارير الثغرات مع الغير
   - احفظ النتائج في مكان آمن

3. **التحديثات:**
   - افحص التحديثات بانتظام
   - استخدم أحدث إصدار من الأداة

🆘 استكشاف الأخطاء:
------------------

**إذا لم تعمل الأداة:**
1. تأكد من تثبيت Python بشكل صحيح
2. تحقق من تثبيت جميع المتطلبات
3. تأكد من اتصال الإنترنت
4. جرب تشغيل الأمر كمسؤول (Windows)

**أخطاء شائعة:**
- `ModuleNotFoundError`: قم بتثبيت المكتبة المفقودة
- `PermissionError`: جرب تشغيل الأمر كمسؤول
- `ConnectionError`: تحقق من اتصال الإنترنت
